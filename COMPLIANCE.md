# Compliance

Allowed:

- Reproduce public solutions with provenance.
- Query Kaggle resources through the CLI.
- Validate adapter structure without loading the base model.
- Package structural-valid `submission.zip`.

Forbidden:

- Multi-account usage.
- Quota bypass.
- Browser automation for login or rule acceptance.
- Hidden-test leakage or leaderboard oracle attacks.
- Fake scores or unverifiable 0.86+ claims.
- Submitting failed validation artifacts.
- Storing `kaggle.json` or tokens in this repository.
